## Installation Instructions
```
curl -LsSf https://astral.sh/uv/install.sh | sh

or

pip install uv
```

in the repo:
```
uv sync [optional flag] --no-cache
```

usage:
```
uv run main.py

or

source .venv/bin/activate
python3 main.py

uv add [python package]
```
